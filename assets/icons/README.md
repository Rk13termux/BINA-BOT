# Icons Directory

This directory contains icon assets for the Invictus Trader Pro app.

## Types:
- `app_icon.png` - Main app icon
- `navigation/` - Bottom navigation icons
- `trading/` - Trading-related icons
- `crypto/` - Cryptocurrency icons

## Sizes:
- 24x24 (small icons)
- 32x32 (medium icons)
- 48x48 (large icons)
- 512x512 (app icon)
